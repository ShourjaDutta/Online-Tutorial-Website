Functions and Pointers in C Programming

Functions in C

Functions in C are blocks of code that perform a specific task. They allow for code reusability and modularity. The syntax for a function is as follows:

return_type function_name(parameters) {
 // code
}

Example

Example of a simple function in C:

#include<stdio.h>
int add(int a, int b) {
 return a + b;
}
int main() {
 int result = add(5, 7);
 printf("Sum: %d", result);
 return 0;
}

Pointers in C

Pointers are variables that store the memory address of another variable. They are a powerful feature in C programming. The syntax for declaring a pointer is as follows:

data_type *pointer_name;

Example

Example of using pointers in C:

 printf("Value of num: %d", *ptr);
 return 0;

Program Questions

1. Write a C program to find the factorial of a number using a function.
2. Create a program to swap two numbers using pointers.

Program Solutions

Solutions for the above questions:

// Program 1:
#include<stdio.h>
int factorial(int n) {
 if (n == 0 || n == 1) {
  return 1;
 } else {
  return n * factorial(n - 1);
 }
}
int main() {
 int num = 5;
 int result = factorial(num);
 printf("Factorial of %d: %d", num, result);
 return 0;
}
// Program 2:
#include<stdio.h>
void swap(int *a, int *b) {
 int temp = *a;
 *a = *b;
 *b = temp;
}
int main() {
 int num1 = 10, num2 = 20;
 printf("Before swap: num1 = %d, num2 = %d\n", num1, num2);
 swap(&num1, &num2);
 printf("After swap: num1 = %d, num2 = %d", num1, num2);
 return 0;
}
