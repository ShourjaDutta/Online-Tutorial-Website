File Handling and Exception Handling in Python Programming

File Handling in Python

File handling allows you to read from and write to files. Python provides built-in functions for file operations.

# Opening a file for reading
file = open("filename.txt", "r")
content = file.read()
file.close()

# Opening a file for writing
file = open("filename.txt", "w")
file.write("Hello, World!")
file.close()

Examples

Example 1: Read content from a file

file = open("sample.txt", "r")
content = file.read()
file.close()
print(content)

Example 2: Write content to a file

file = open("output.txt", "w")
file.write("This is a sample text.")
file.close()

Exception Handling in Python

Exception handling allows you to handle errors and exceptions in a graceful manner, preventing your program from crashing.

try:
  # Code that may raise an exception
except ExceptionType as e:
  # Code to handle the exception

Program Questions

1. Write a Python program to copy the content of one file to another.

2. Implement exception handling to handle a FileNotFoundError while reading a file.

Program Solutions

1. Write a Python program to copy the content of one file to another.

with open("source.txt", "r") as source_file:
  with open("destination.txt", "w") as destination_file:
   destination_file.write(source_file.read())

2. Implement exception handling to handle a FileNotFoundError while reading a file.

try:
  with open("nonexistent_file.txt", "r") as file:
   content = file.read()
except FileNotFoundError as e:
  print("File not found:", e)
