Variables and Data Types in C Programming

Syntax for Variables

In C programming, a variable is a named location used to store data. The syntax for declaring a variable is as follows:

data_type variable_name;

Examples

Example of declaring and initializing variables:

int age = 25;
float temperature = 98.6;
char grade = 'A';

Data Types in C

C supports various data types such as int, float, char, double, etc. Here are some examples:

int count = 10;
float price = 24.99;
char initial = 'M';

Program Questions

1. Write a C program to calculate the area of a rectangle using variables.
2. Create a program to convert temperature from Celsius to Fahrenheit.

Program Solutions

Solutions for the above questions:

// Program 1:
#include<stdio.h>
int main() {
 int length = 5, width = 8;
 int area = length * width;
 printf("Area of the rectangle: %d", area);
 return 0;
}
// Program 2:
#include<stdio.h>
int main() {
 float celsius = 30;
 float fahrenheit = (celsius * 9/5) + 32;
 printf("Temperature in Fahrenheit: %f", fahrenheit);
 return 0;
}
