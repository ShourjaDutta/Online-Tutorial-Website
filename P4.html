Control Flow and Loops in Python Programming

if-else Statement

The if-else statement is used for decision-making in Python. It allows the program to execute different blocks of code based on a specified condition.

if condition:
  # Code to be executed if the condition is True
else:
  # Code to be executed if the condition is False

Examples

Example 1: Check if a number is even or odd

num = 10
if num % 2 == 0:
  print("Even")
else:
  print("Odd")

Example 2: Determine if a person is eligible to vote

age = 18
if age >= 18:
  print("Eligible to vote")
else:
  print("Not eligible to vote")

Program Questions

1. Write a Python program to find the maximum of two numbers using if-else.

2. Check if a given year is a leap year or not.

Program Solutions

1. Write a Python program to find the maximum of two numbers using if-else.

num1 = 15
num2 = 20
if num1 > num2:
  print("Maximum:", num1)
else:
  print("Maximum:", num2)

2. Check if a given year is a leap year or not.

year = 2024
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
  print("Leap year")
else:
  print("Not a leap year")
