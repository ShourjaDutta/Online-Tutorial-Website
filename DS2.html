Arrays in Data Structures Tutorial

Syntax for Array Declaration

An array is a collection of elements of the same type stored in contiguous memory locations. The syntax for declaring an array is as follows:

data_type array_name[array_size];

Example

Example of using an array in C:

#include<stdio.h>
int main() {
 int numbers[5] = {1, 2, 3, 4, 5};
 printf("First element: %d", numbers[0]);
 return 0;
}

Program Questions

1. Write a C program to find the sum of elements in an array.
2. Create a program to find the largest element in an array.

Program Solutions

Solutions for the above questions:

// Program 1:
#include<stdio.h>
int main() {
 int numbers[5] = {1, 2, 3, 4, 5};
 int sum = 0;
 for (int i = 0; i < 5; i++) {
  sum += numbers[i];
 }
 printf("Sum of elements: %d", sum);
 return 0;
}
// Program 2:
#include<stdio.h>
int main() {
 int numbers[5] = {10, 5, 8, 15, 7};
 int max = numbers[0];
 for (int i = 1; i < 5; i++) {
  if (numbers[i] > max) {
   max = numbers[i];
  }
 }
 printf("Largest element: %d", max);
 return 0;
}
