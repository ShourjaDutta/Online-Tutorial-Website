Variables and Data Types in Python Programming

Syntax for Variable Declaration

In Python, variables are dynamically typed. You can assign a value to a variable without explicitly declaring its type.

variable_name = value

Examples

Example 1: Declare and print an integer variable

age = 25
print(age)

Example 2: Declare and print a string variable

name = "John"
print(name)

Program Questions

1. Declare a float variable named "price" with a value of 19.99.

2. Create a variable "is_python_fun" and assign it a boolean value.

Program Solutions

1. Declare a float variable named "price" with a value of 19.99.

price = 19.99

2. Create a variable "is_python_fun" and assign it a boolean value.

is_python_fun = True
