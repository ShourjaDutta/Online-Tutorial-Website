Control Structures in C Programming

If Statement

The if statement is used to make decisions in C programming. Its syntax is as follows:

if (condition) {
 // code to be executed if the condition is true
}

Example

Example of using an if statement:

#include<stdio.h>
int main() {
 int num = 10;
 if (num > 0) {
  printf("The number is positive.");
 }
 return 0;
}

Program Questions

1. Write a C program to check whether a number is even or odd using an if statement.
2. Create a program to determine if a student has passed or failed based on their exam score.

Program Solutions

Solutions for the above questions:

// Program 1:
#include<stdio.h>
int main() {
 int num = 15;
 if (num % 2 == 0) {
  printf("The number is even.");
 } else {
  printf("The number is odd.");
 }
 return 0;
}
// Program 2:
#include<stdio.h>
int main() {
 int score = 75;
 if (score >= 50) {
  printf("Congratulations! You passed.");
 } else {
  printf("Sorry, you failed.");
 }
 return 0;
}
